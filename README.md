# Workout App

A simple iOS app to track daily workouts.


### Notice:
Initially, the Workout screen will appear empty. Once you add a workout log, the data will be displayed in the table view. Until then, the screen will remain empty.


### Features

* Add workout details
* Start/Stop timer
* Enter distance and notes
* Save workouts using Core Data
* View workout history
* Delete workouts
* View active minutes and calories burned


### Built With

* Swift
* UIKit
* Core Data
* Xcode
